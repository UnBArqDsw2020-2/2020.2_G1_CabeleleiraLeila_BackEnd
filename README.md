# 2020.2_G1_CabeleleiraLeila_BackEnd


## Requisitos 

* Java 11
* Maven 3.6.3

## Execução

./mvnw spring-boot:run 

## Swagger

Acesso http://localhost:8080/api/swagger-ui/

# Usando o docker e docker-compose:

linux: 

```console
docker-compose up --build
```

## Prettier-Java
Instalação 
  ```
  npm install -g prettier prettier-plugin-java
  ```

Execução
  ```
  prettier --write "**/*.java"
  ```
